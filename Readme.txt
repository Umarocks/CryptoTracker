# Crypto Price Tracker

[DEMO](https://umarocks.github.io/CryptoTracker/)

## Description

Crypto Price Tracker is a simple web application for monitoring cryptocurrency prices in real-time. It provides users with up-to-date information on various cryptocurrencies, including their current prices and historical data.

## Features

- Real-time cryptocurrency price updates.
- Historical price charts for selected cryptocurrencies.
- User-friendly interface for easy navigation.
- Customizable watchlist to track your favorite cryptocurrencies.
- Price alerts for specified cryptocurrency price thresholds.

## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/Umarocks/CryptoTracker
   ```

2. Install dependencies:

   ```bash
   cd crypto-price-tracker
   npm install
   ```

3. Start the app:

   ```bash
   npm start
   ```

4. Open your web browser and go to `http://localhost:3000` to launch the app.

## Usage

1. Visit the app in your web browser.

2. Browse and select your preferred cryptocurrencies from the list.

3. Click on a cryptocurrency to access real-time price updates and historical charts.
